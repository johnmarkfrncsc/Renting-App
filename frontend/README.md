# Rentara - Property Management App

A full-stack property management web application built with React, Node.js, Express, and MongoDB.

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, React Router
**Backend:** Node.js, Express, MongoDB, Mongoose
**Auth:** JWT, Google OAuth
**Storage:** Cloudinary

## Getting Started

### Prerequisites

- Node.js
- MongoDB Atlas account
- Cloudinary account
- Google Cloud Console project

### Installation

1. Clone the repository

```bash
git clone https://github.com/johnmarkfrncsc/Renting-App.git
cd Renting-App
```

2. Setup backend

```bash
cd backend
npm install
cp .env.template .env
# fill in your values in .env
```

3. Setup frontend

```bash
cd frontend
npm install
cp .env.template .env
# fill in your values in .env
```

4. Run the app

```bash
# backend
cd backend
npm run start

# frontend
cd frontend
npm run dev
```

## Environment Variables

See `backend/.env.template` and `frontend/.env.template` for required variables.
